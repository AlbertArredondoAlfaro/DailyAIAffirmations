# Daily Affirmations

Simple iOS app that delivers a daily positive affirmation, with widget support and shareable visuals.

## Goals (MVP)
- Daily affirmation based on date
- Random button and custom placeholders
- Home + Lock Screen widget
- Share as image
- Ads (AdMob) with IAP to remove ads
- Spanish/English based on Locale

## Tech
- SwiftUI (iOS 26)
- Liquid Glass design language
